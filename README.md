# blockmf
Block Matrix Factorization 

Block based matrix factorization approach that splits a given matrix into required number of blocks and factorization achieved on individual blocks


## Getting Started


### Prerequisites

```
Python 3.0
CUDA v8.0
pyCUDA v1.8

The GPU based Block matrix factorization has been developed using python using pyCUDA support.

```

### Installing


```
git clone 'https://github.com/17mcpc14/blockgmf'

```
